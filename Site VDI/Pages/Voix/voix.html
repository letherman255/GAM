Voix


Présentation:


Le VDI est fourni avec 2 téléphones IP. Il faut activé le protocole SIP (Session Initiation Protocol) dans le VDI pour utiler le service associé aux téléphones. Depuis 2007, le SIP est le plus utilisé pour la téléphonie IP.

I) Le protocole SIP:


Le protocole SIP est un protocole standard ouvert (format libre) qui permet la gestion de sessions utilisé dans les télécommunications multimédia (son, image, ...). Afin de l'activer:
DidaVDI -> Service voix -> Démarrage SIP pbx
.

II) Téléphone logiciel:


Il s'agit d'un téléphone logiciel, c'est-à-dire sur ordinateur. Il existe un équivalent que beaucoup connaissent: Skype.



1) Fonctionnement:


Pour appeler le téléphone, on a juste a entrer son numéro et appuyer sur la touche appeler.
