Données

Présentation:


Le système VDI est équipé d'un disque SSD (Solid State Drive) de 60 GO sur lequel est stocké son OS (Operating System) ainsi que divers fichier comme des vidéos, de la musique, des pages web mais aussi son mode d'emploi. Pour communiquer certaine de ces données le serveur VDI utilise plusieurs protocoles.

I) Le protocole HTTP


Le protocole HTTP (HyperText Transfer Protocol) est un protocole permettant la diffusion de page web le réseau comme cette page ou encore la page de configuration du VDI. On peut aussi transmettre de fichier de toute sorte mais il ne serons probablement pas lisible par votre navigateur.

II) Le protocole FTP


Le protocole FTP (File Transfert Protocol) est un protocole permettant de visualiser transferer et manipuler les fichier d'un serveur à distance. Une variante de ce protocole est le SFTP (Secure File Transfert Protocol), cette variante est plus sécurisée que le FTP.


III) Le protocole SSH


Le SSH (Secure Shell) est un protocole que le VDI ne gère pas mais il convient de le citer car il est essentiel dans la gestion à distance d'un serveur, il permet de litéralement piloter (en invite de commande bien entendu) un serveur distant comme une console à distance.
