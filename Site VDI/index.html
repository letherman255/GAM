Accueil

Le serveur VDI est un serveur éducatif composé d'une armoire rack dans laquelle vient se loger un routeur, un switch (à nne pas confondre avec un hub) un tiroir pour ranger les cables et le serveur, serveur qui fonctionne sur une distribution de Linux. Le tout est surmonté d'un SetTopBox (un petit ordinateur tout juste capable de lire une vidéo ou une page web) et d'un écran pour celui-ci. Sous le VDI, une armoire de rangement est là pour accueillir les 2 téléphones ip, la radio ip, un set clavier/souris et un ordinateur fonctionnant sous Debian (Linux).
